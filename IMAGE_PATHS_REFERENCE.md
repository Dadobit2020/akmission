# Image Paths Reference for Addis Kidan Mission Website

## Important Notes
- All folder names with spaces need URL encoding (%20) in HTML/CSS
- Example: "AKEC Mission team 2025 Mission Trip" becomes "AKEC%20Mission%20team%202025%20Mission%20Trip"
- Use `_large.jpeg` for hero/background images
- Use `_medium.jpeg` for card/grid images
- Use `_small.jpeg` or `_thumbnail.jpeg` for thumbnails

## CSS Background Image Syntax

```css
/* With transparent overlay */
background-image: linear-gradient(135deg, rgba(217, 119, 6, 0.7) 0%, rgba(146, 64, 14, 0.7) 100%), 
                  url('../images/FOLDER%20NAME/subfolder/image_large.jpeg');
background-size: cover;
background-position: center;
```

## HTML img Tag Syntax

```html
<!-- For spaces in folder names -->
<img src="../images/Medical%20Team/optimized_images_20260119_193651_236998/hospital/hospital_large.jpeg" 
     alt="Description" 
     class="img-fluid">
```

## Overlay Colors
- **Primary Overlay:** `rgba(217, 119, 6, 0.7)` - Orange
- **Secondary Overlay:** `rgba(146, 64, 14, 0.7)` - Brown
- **Dark Overlay:** `rgba(0, 0, 0, 0.5)` - Black (for lighter images)
- **Light Overlay:** `rgba(255, 255, 255, 0.3)` - White (for darker images)

## Image Size Guidelines
- **Hero/Background:** 1920x1080 or larger (use _large or _xlarge)
- **Cards/Grid:** 800x600 (use _medium)
- **Thumbnails:** 400x300 (use _small or _thumbnail)
- **Full Width Sections:** Use _large or _xlarge
- **Sidebar Images:** Use _medium or _small

## Notes
- All images are optimized and available in multiple sizes
- Always use URL encoding (%20) for spaces in paths
- Add transparent overlays to ensure text readability
- Match image content to section purpose
- Test all image paths after implementation
